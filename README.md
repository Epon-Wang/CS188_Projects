# CS188 Projects

Implementations of Projects 1-4 from UCB CS188 Introduction to Artificial Intelligence

To review a project
```bash
cd ~/CS188_Projects/<project_name>
```
Available Projects

| `<project_name>` | Description |
|-------|-------------|
|`search`   |[Project 1 Search](search/readme.md)|
|`reinforcement`   |[Project 3 Reinforcement Learning](reinforcement/readme.md)|

To check the overall results of this project
```bash
python autograder.py
```

To check the results of a question (ex. q2) in this project
```bash
python autograder.py -q q2
```
